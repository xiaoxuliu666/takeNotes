# Floating Notes

一个小型桌面浮窗笔记/待办应用。

## 功能

- 多栏目管理笔记和待办
- 每个栏目内笔记和待办分区隔离，互不混在同一个列表里
- 支持跨所有栏目、笔记和待办的全局模糊搜索
- 内容中的 `http://` 和 `https://` 链接会自动变成可点击超链接
- 窗口可拖动、可调整大小
- 可切换固定到桌面最顶层
- 可切换开机自启
- 可设置笔记数据存储文件夹，并保存常用笔记文件夹用于快速切换
- 可迁移笔记到新文件夹；迁移成功后会清空旧文件夹中的笔记数据
- 输入、编辑、勾选、删除都会实时保存到本地 JSON 文件
- 关闭应用、关机、再次打开后数据仍会保留

## 运行

```bash
npm install
npm start
```

默认数据保存在 Electron 的用户数据目录中，文件名为 `notes-store.json`。也可以在应用内通过存储设置切换到其他文件夹。
